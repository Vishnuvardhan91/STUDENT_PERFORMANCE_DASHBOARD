## Student Performance Dashboard
## Overview

This project is a basic Student Performance Dashboard created using Microsoft Power BI.
It shows student data like marks and attendance in an easy-to-understand way.

## Data

The dataset includes:

Student Name
Subject
Marks
Attendance
Teacher Name
## Features
View student performance
Subject-wise marks analysis
Teacher name included
Simple charts and visuals
## Tool Used
Microsoft Power BI
## How to Use
Open the .pbix file
View the dashboard
Use filters if available
